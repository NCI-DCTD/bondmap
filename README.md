# BondMap

A Python-based cheminformatics pipeline designed to identify unknown compounds in mixtures by integrating **Infrared (IR) Spectroscopy** with **Mass Spectrometry (MS)** data.

## ⚙️ Installation

### Requirements

- Python v 3.11.5 or greater

- Jupyter notebook (optional)

1. **Clone the repository:**
   ```bash
   git clone https://github.com/NCI-DCTD/bondmap.git
   cd bondmap

2. **Install supporting packages:**
   ```bash
   pip install -r requirements.txt

3. **Install jupyter notebook**
   ```bash
   pip install notebook
   jupyter notebook source/bondmap.ipynb
